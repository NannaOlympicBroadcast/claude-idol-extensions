## 🖱️ 鼠标操作入口

agent skill 支持通过鼠标与偶像人格进行交互。以下是可用的鼠标操作及其说明：

| 操作 | 触发方式 | 说明 |
|------|----------|------|
| 唤醒偶像 | 单击偶像头像 | 激活当前选中的偶像 skill，进入对话模式 |
| 切换偶像 | 双击偶像头像 | 在已安装的偶像 skill 之间循环切换 |
| 查看简介 | 右键单击偶像头像 | 弹出该偶像 skill 的人格简介与使用说明 |
| 拖动偶像 | 按住并拖拽头像 | 将偶像悬浮窗移动到屏幕任意位置 |
| 收起偶像 | 滚轮向下滚动 | 将偶像悬浮窗最小化到角落 |
| 展开偶像 | 滚轮向上滚动 | 将偶像悬浮窗从最小化状态恢复 |

> **提示**：鼠标悬停在偶像头像上时，会显示该偶像的当前情绪状态及一句随机台词。
